## What this folder is

This folder contains **day-wise, session-wise work logs**. Each work day gets a single file named:

- `DD-MM-YYYY-work.md` (example: `09-04-2026-work.md`)

## How we will use it (rule)

- **One file per day**
- **Append-only throughout the day** (never overwrite earlier entries)
- **Session-wise entries**: each time we work (a new chat/session), we add a new “Session” block for that day

## What each daily file should contain

- **Date + timezone**
- **Session index** and **time range** (start/end if known)
- **Goals / scope**
- **Work completed** (bullet list)
- **Files changed / added** (paths)
- **Commands run** (if any)
- **Notes / follow-ups** (bugs, TODOs, decisions)
